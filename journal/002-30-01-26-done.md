## Fix FOUC: make dark theme the CSS default

Eliminated light-to-dark flash on page load by swapping the CSS theme structure. `:root` now defines dark theme colors (zero-JS default), `[data-theme="light"]` is the override. FOUC script only acts for light-mode users. All `[data-theme="dark"]` selectors across the codebase flipped to `[data-theme="light"]`.

~ src/styles/theme.css
~ src/layouts/BaseLayout.astro
